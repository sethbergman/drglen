### Setup

```
git clone https://github.com/vipcare/drglen.git && cd drglen
```

### Install dependencies

```sh
npm i # or use yarn
yarn
```

### Start the app

```sh
npm start
# with yarn
yarn start
```

##### Visit [http://0.0.0.0:5000](http://0.0.0.0:5000) in your browser.
